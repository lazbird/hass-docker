Place brand images for this integration in this directory.

Required for custom branding in Home Assistant 2026.3+:

- `logo.png`: Used on the integration/auth/config pages
- `icon.png`: Used for the integration icon

Optional dark mode variants:

- `dark_logo.png`
- `dark_icon.png`

Notes:

- Keep the Tuya SDK and login flow unchanged; only the local integration domain and
  branding are customized here.
- After adding the images, restart Home Assistant and refresh the browser cache.
